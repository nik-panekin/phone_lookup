Данный скрипт осуществляет поиск отзывов (негативных или позитивных) для
пользователя указанного телефонного номера. Технически это реализуется
путём обращения к базе сайта https://www.telefonnyjdovidnyk.com.ua/.

В настоящей версии программы поддерживаются только украинские номера.

Формат запуска:
    python phone_lookup.py

Зависимости скрипта перечислены в файле requirements.txt.

Версия интерпретатора Python - 3.8.10.

Используйте и/или модифицируйте данный программный код без ограничений.
